Atalhos usados
---------

- Cmd (⌘) – the Command Key symbol
- Ctrl (⌃) - the Control Key symbol
- Option (⌥) – the Option Key symbol
- Shift (⇧) – the Shift Key symbol
- Esc (⎋) – the Escape Key symbol
- Return (↵) – the Return symbol
- Delete (⌫) – the Delete / Backspace symbol
